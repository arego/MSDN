Naming Styles

There are no naming styles defined for XML elements. But here are some commonly used:

Style Example Description
Upper case FIRSTNAME All letters upper case
Underscore first_name Underscore separates words
Pascal case FirstName Uppercase first letter in each word
Camel case firstName

Uppercase first letter in

each word except the first

If you choose a naming style, it is good to be consistent!

XML documents often have a corresponding database. A good practice is to use naming rules of your database for the elements in the XML documents.

There are 5 predefined entity references in XML

&lt; < less than
&gt; > greater than
&amp; & ampersand
&apos; ' apostrophe
&lt;&quot; " quotation mark
